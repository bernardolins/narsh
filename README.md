# NARSH

## Getting Started
```bash
# compiles all commands
make all
# compiles just ls
make ls
# compiles just rm
make rm
# compiles just head
make head
# compiles just tail
make tail
```
## Running
Binaries are compiled into `bin/*` directory. Just run as if it's a linux command

## Available options

```bash
bin/ls [-l -a]
bin/rm [-f]
bin/tail [-n]
bin/head [-n -c]
```
